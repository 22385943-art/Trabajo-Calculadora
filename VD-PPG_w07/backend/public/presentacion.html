Calculadora Estadística

Frontend y Backend unidos para mostrar los calculos de forma visual

01. EL PROYECTO

¿En qué consiste la aplicación?

Funcionalidad y Experiencia de Usuario

La web es una herramienta de diagnóstico estadístico que permite a cualquier usuario transformar datos brutos en conocimiento visual sin necesidad de software especializado. Sus funciones principales son:

  • Procesamiento de datos: Interfaz capaz de interpretar estructuras CSV complejas.
  • Análisis bajo demanda: El usuario decide qué variables comparar (Independiente vs Dependiente).
  • Feedback inmediato: Cálculo instantáneo de la recta de regresión y su bondad de ajuste.
02. ESTRUCTURA TÉCNICA

Arquitectura Cliente-Servidor

Separación de Lógica y Presentación

Para garantizar un sistema escalable y profesional, hemos separado la aplicación en dos grandes bloques comunicados por JSON:

  • Backend (Node.js): Actúa como el "cerebro". No solo sirve los archivos, sino que realiza las operaciones matemáticas pesadas, protegiendo la lógica del negocio.
  • Frontend (Client-side): Una SPA (Single Page Application) que gestiona el estado de los datos cargados mediante JavaScript, permitiendo cambios dinámicos sin recargar la página.
03. FLEXIBILIDAD DE ENTRADA

Fuentes de Datos

Dataset Local vs. API Remota

La aplicación ha sido diseñada para ser versátil, permitiendo al analista trabajar con información proveniente de dos entornos distintos:

  • Carga de Dataset (CSV): Permite el procesamiento de archivos locales almacenados en el equipo del usuario. El sistema interpreta la estructura de filas y columnas de forma automática tras la subida.
  • Conexión a API (JSON): Facilita la importación de datos en tiempo real desde servidores externos. El usuario solo necesita proporcionar la URL del endpoint para que la aplicación consuma y formatee los datos.

Ambos métodos convergen en un mismo formato interno, garantizando que el análisis de regresión sea idéntico sin importar el origen de la información.

04. MOTOR MATEMÁTICO

Métricas de Regresión

Fórmulas y Análisis de Precisión

El backend procesa los datos para devolver tres indicadores fundamentales que definen el modelo:

Ecuación de la Recta

Y = β0 + β1X

Define la relación lineal entre las variables.

Bondad de Ajuste

R2

Indica el porcentaje de varianza explicada por el modelo.

Precisión

σest

Mide la dispersión de los datos respecto a la recta.

  • β1 (Pendiente): Representa el cambio en Y por cada unidad de X.
  • β0 (Intercepto): El valor de Y cuando X es igual a cero.
  • Fiabilidad: Un R2 cercano a 1 indica una predicción altamente confiable.
05. RESULTADO VISUAL

Gráficos con Plotly.js

Interactividad y Análisis Visual

La visualización es el último paso de la ingeniería inversa aplicada al proyecto:

  • Renderizado Reactivo: El gráfico se limpia (purge) y se redibuja automáticamente al cambiar las variables X o Y.
  • Dualidad de Datos: Se muestran simultáneamente los puntos reales (Scatter) y la tendencia teórica (Línea de regresión).
  • Diseño Responsivo: El motor visual se ajusta dinámicamente al tamaño de la pantalla del dispositivo.
