PROYECTO AEC1

Analizador de Regresión Lineal

Herramienta de Ingeniería para el Análisis de Datos

Desarrollo Full-Stack: Node.js + Vanilla JS

Usa los botones para navegar

01. ARQUITECTURA

Estructura del Sistema

El proyecto se divide en tres capas fundamentales siguiendo el principio de separación de responsabilidades:

  • Frontend: Interfaz reactiva construida con HTML5, CSS Grid y JavaScript asíncrono.
  • Backend: Servidor en Node.js encargado del procesamiento pesado y la lógica matemática.
  • API: Comunicación mediante JSON para el intercambio de datos y resultados.
02. ADQUISICIÓN DE DATOS

Flexibilidad de Entrada

Implementamos dos métodos de ingesta de datos mediante Ingeniería Inversa:

  • Carga Local: Procesamiento de archivos CSV mediante FormData.
  • Consumo de API Externa: El backend actúa como proxy para evitar problemas de CORS, permitiendo analizar datos de cualquier URL pública.
03. EL BACKEND

Procesamiento y Regresión

La lógica no reside en el cliente para garantizar eficiencia:

  • Cálculo de Pendiente (β1) e Intercepto (β0).
  • Obtención del Coeficiente de Determinación (R²) para medir la fiabilidad del modelo.
  • Cálculo del Error Estándar del análisis.
04. VISUALIZACIÓN

Interfaz Plotly.js

Transformación de números en información visual accionable:

  • Gráfico de dispersión (Scatter Plot) con los datos reales.
  • Superposición de la Línea de Mejor Ajuste calculada.
  • Interfaz interactiva que permite explorar los puntos de datos.
05. CONCLUSIÓN

Ingeniería de Software Aplicada

Este proyecto demuestra la capacidad de integrar:

  • Manejo de estados en el cliente (State Management).
  • Diseño responsivo y profesional (UX/UI).
  • Consumo y transformación de datos en tiempo real.
